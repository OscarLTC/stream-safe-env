# Stream Safe Env

Hide sensitive environment variables when streaming or recording your screen.

## Features

- **Toggle masking** of `.env` file values with a single click
- **Automatic detection** of `.env`, `.env.local`, `.env.development`, etc.
- **Persistent state** - remembers your preference across sessions
- **Visual indicator** in status bar

![Demo](images/demo.gif)

## Usage

1. Open any `.env` file
2. Click the status bar button to toggle masking on/off
3. Masked values appear as `••••••••`

## Commands

- `Toggle Env Masking` - Enable/disable variable masking

## Requirements

VS Code 1.106.0 or higher
